# FPSemana04

Exercício: ler um ficheiro JSON com um script Python `read_json.py`.
Mensagens de erro esperadas:
- Erro: Ficheiro Não Encontrado!
- Erro: Ficheiro Não Contém JSON Válido!
- Erro: Ficheiro Vazio!
- Erro: Campos Obrigatórios em Falta!

No final deve sempre imprimir: "Processo Concluído!".
